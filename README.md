# SAED_UFFS
## Para verificar atualizações  
abrir terminal e digitar:
```
    make update
```

## Para rodar o programa  
  abrir terminal e digitar:  
```
    cd ~/SAED_UFFS [ENTER]
    sudo python3 software.py [ENTER]
```

## Instalar dependencias no Ubuntu/Debian
  abrir terminal e digitar:  
```
    make install-deb
```


## Instalar dependencias no Fedora
  abrir terminal e digitar:  
```
    make install-fedora
```
